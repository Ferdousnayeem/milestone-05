DOM হলো ইন্টারনেট ব্রাউজার দ্বারা চালিত Document Object Model. জাভাস্ক্রিপ্টে DOM বিভিন্ন প্রোগ্রাম দ্বারা ডকুমেন্টের স্ট্রাকচার, স্টাইল সহ আরো অন্যান্য অনেক কিছু পরিবর্তন এবং পরিবর্ধন করা যায়। DOM তার ডকুমেন্টকে অনুসারে ‍node এবং object রূপে উপস্থাপন করে।

ওয়েব ব্রাউজারের ডকুমেন্ট থেকে পাঁচ উপায়ে HTML elements সিলেক্ট করা যায়। যেমন: 1) getElementsByTagName() 2) getElementsByClassName() 3) getElementById() 4) querySelector() 5) querySelectorAll()

যদি কোনো child element কে trigger করা হয় তাহলে তার parent element গুলোও ঐ trigger এর মধ্যে পড়ে যায়। এই অবস্থাকে Event bubble বলা হয়। তবে চাইলে event.stopPropagation() এর মাধ্যমে parent element এর trigger হওয়াকে বন্ধ করা যায়।

জাভাস্ক্রিপ্টে যখন একটি ফাংশনের প্যারামিটার হিসাবে আরেকটি ফাংশনকে পাঠানো হয়, তখন ঐ ফাংশনকে callback function বলে।
